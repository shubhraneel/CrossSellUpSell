# CrossSellUpSell

## Running Instructions

1. Start virtual environment

   ```bash
   python3 -m venv env
   ```

2. Install requirements

   ```bash
   pip install -r requirements.txt
   ```

3. Start the server

   ```bash
   python3 app.py
   ```

4. Open [localhost:5000](http://localhost:5000) in a browser

## Local Requirements

* Python 3.85 or above
* pip
